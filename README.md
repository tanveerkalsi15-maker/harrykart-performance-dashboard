# HarryKart Performance Dashboard

## Project Overview
An e-commerce sales performance dashboard built using Power BI to analyze revenue, orders, products, and customer insights.

## Tools Used
- Power BI
- Excel / CSV
- DAX
- Data Cleaning

## Dashboard Features
- Total Revenue Analysis
- Average Order Value
- Top Performing Products
- Sales Trends
- Category Performance

## Key Insights
- Identified top revenue-generating products
- Analyzed sales patterns
- Compared category performance

## Files
- harrykart_data.csv → Dataset
- HarryKart_Dashboard.pbix → Power BI Dashboard
